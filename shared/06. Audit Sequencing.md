# 06. Audit Sequencing -post-refactor playbook

Mirrors Phase 16 ALM playbook, scaled for tri-surface scope (peripheral + alm + dex). Goal: line ≥ 3-eye coverage; exit gate = 2 successive zero-finding rounds.

## Scope partitioning

Three audit lots, parallelized:

| Lot | Surface | LOC | Priority |
|---|---|---:|---|
| L1 | `peripheral/src/{access,oracle,errors,libs,factory,math}` | ~1,400 | Highest -root-of-trust for both alm + dex |
| L2 | alm post-refactor (`Vault`, `adapters/Dex`, `adapters/Swapper`, `Factory`) | ~1,500 | High |
| L3 | dex post-refactor (top-level + modules + libraries) | ~10,000 | High, larger |

Each lot reviewed by **4 auditors in parallel** per round. Auditors blind-paired (no inter-aud comms during round). Findings reconciled at round-end.

## Round structure

### Round 0 -Internal pre-audit (1 week)

- Static: Slither, Mythril, Aderyn across all 3 lots.
- Manual: dev team self-review using `audit-findings.md` checklist (carry-forward from Phase 16 ALM).
- Fuzz: Foundry invariant suite -peripheral (AC rotation, PriceProvider feed timelock, LibTimelock pack/unpack), alm (Vault solvency + HWM), dex (PoolStorage invariants per module).
- **Gate**: zero static high-severity; >90% line coverage; invariant suite green ≥10M runs.

### Round 1 -External audit firms × 2, lots L1 + L2 (3 weeks)

- L1 (peripheral): both firms → highest density (smallest, most-trust-loaded).
- L2 (alm): one firm primary + one secondary cross-check.
- L3 (dex): defer to Round 2 (firms onboarding dex code in parallel).
- Findings rolled into `audit-findings.md` w/ severity (Crit/High/Med/Low/Info) + status (Open/Fixed/Acknowledged/Defer).
- **Gate**: zero Crit; all High fixed or formally Acknowledged with rationale.

### Round 2 -External firms × 2, lot L3 (4 weeks)

- Both firms full L3 dex review.
- Re-review L1 + L2 fixes from Round 1 (scope: only diff since R1).
- **Gate**: zero Crit; all High fixed.

### Round 3 -Cross-fix re-audit + competition (2 weeks)

- All Round 1 + Round 2 fixes re-audited.
- Public competition phase (Code4rena / Cantina / Sherlock) for L1 + L2 (cheaper at this size); private for L3.
- Findings dedup'd vs prior rounds.
- **Gate**: zero new Crit/High; ≤3 unique Med across all lots.

### Round 4 -Final pass (1 week)

- Same 2 firms revisit ALL changes since Round 1 origin.
- **Exit condition**: zero findings of Med+ severity.
- If non-zero → loop back to fix + Round 5 (max 2 successive zero rounds enforces termination).

### Round 5+ (only if Round 4 produced findings)

Repeat fix → re-audit cycle. Exit on **2 successive zero-finding rounds**.

## Defer list management

Findings explicitly deferred (e.g. EIP-170 split for AdminV1) tracked in `defer-list.md` w/:
- Rationale
- Compensating control
- Target phase for resolution (e.g. Phase 42C)
- Sign-off: (gov-lead, lead-auditor, eng-lead)

Defer list ≤ 5 items at exit. Each deferred item linked from `audit-findings.md`.

## Coverage requirements

- **Line**: ≥ 3-eye on every line of L1; ≥ 2-eye on L2; ≥ 2-eye on L3 critical paths (entry fns, modifier checks, revert sites).
- **Branch**: ≥ 95% per Foundry coverage report; uncovered branches require comment justifying unreachability.
- **Invariant**: ≥ 10M runs/invariant; 100M for AC rotation + Vault solvency.

## Auditor mix

Per round: 4 auditors split:
- 1 senior (≥5 yr Solidity audit, prior DeFi protocol)
- 2 mid (3+ yr, AMM or vault domain)
- 1 specialist (oracle / MEV / cross-chain -rotates per lot)

Specialist for L1: oracle expert (PriceProvider Chainlink + sequencer guard).
Specialist for L2: vault / 4626 / 7540 expert.
Specialist for L3: AMM math + diamond-pattern expert.

## Sign-off matrix

| Artifact | Sign-off required from |
|---|---|
| `audit-findings.md` final | gov-lead, eng-lead, all 4 firms |
| Defer-list final | gov-lead + 1 firm |
| Mainnet deploy proposal | gov multisig (M-of-N), all sign-offs above |

## Mainnet deploy gate

- All 4 lots complete Round 4 w/ zero Med+ findings.
- Defer-list signed off.
- Testnet redeploy ran ≥ 2 weeks w/ keeper + front e2e green.
- Bug bounty active ≥ 1 week pre-mainnet w/ no Crit/High submissions.
- Gov multisig timelock proposal queued (peripheral AC rotation, factory bootstrap).

## Bug bounty post-deploy

Immutable Bug Bounty tier:
- Crit: $250k–$1M
- High: $50k–$150k
- Med: $5k–$25k
- Low: $500–$2.5k

Scoped to peripheral + alm + dex. Out-of-scope: known-deferred items, third-party deps (Solady, LiFi, Chainlink, LZ).
