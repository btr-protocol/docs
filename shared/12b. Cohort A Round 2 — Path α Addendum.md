# 12b. Cohort A Round 2 -Path α Addendum (Phase 42B.1.6)

Addendum to `12. Cohort A Round 2.md`. Resolves the Governance 9.2 < 9.5 blocker via Path α (non-breaking dex AC bridge).

## 1. What landed

Single commit at `~/Work/btr/dex` (evm subtree). Net LOC: src +44 / -3, +1 new minimal interface (12 LOC), +1 new test file (171 LOC, 4 tests).

### Code changes

- **`src/interfaces/IPool.sol`** -appended `address ac` to `PoolStorage` (last field). ERC-7201 namespaced storage append-only invariant preserved; pre-existing `modules` mapping at offset 13 untouched.
- **`src/interfaces/external/IAccessControl.sol`** *(new, 12 LOC)* -minimal local interface declaring only `owner() view returns (address)`. Local rather than peripheral import to keep dex single-selector consumer; full peripheral surface adoption deferred to Path β.
- **`src/modules/Base.sol`** -`onlyOwner` now calls new internal `_owner()` helper. `_owner()` returns `IAccessControl(_s().ac).owner()` when `ac != 0`, else `_s().owner` (legacy fall-through).
- **`src/modules/Admin.sol`** -`onlyOwner` override updated to use `_owner()`. New external `setAc(address)` (gated by current `onlyOwner`, idempotent) and `getAc() view`. Emits `IAdminConfig.AcUpdated`.
- **`src/modules/Distributor.sol`** -sole inline `_s().owner` check (`finalizePointsCampaign`) re-routed through `_owner()` for symmetry.
- **`src/interfaces/modules/IAdmin.sol`** -`IAdminConfig` extended with `setAc` / `getAc` selectors and `AcUpdated` event.

### Test additions

`test/PathAlphaACTest.t.sol` -4 new tests, all green:

1. `test_legacyPath_ownerEnforced` -`ac=0` ⇒ `$.owner` enforced; attacker rejected with `Ownable.Unauthorized`.
2. `test_singletonPath_atomicRotation` -`ac=set` ⇒ singleton AC `owner()` enforced; rotating the AC's owner via 1 tx flips authorization on **N pools simultaneously** (verified across 2 pools).
3. `test_setAc_gatedAndReversible` -`setAc` is `onlyOwner`-gated; resetting `ac=0` returns to legacy path with original owner re-authorized.
4. `test_storageLayout_acAppended` -direct slot probe confirms `ac` lives at an offset strictly greater than the modules mapping (offset 13), proving append-only growth (i.e. no field reorder).
