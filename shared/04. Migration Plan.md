# 04. Migration Plan -Phase 42B

Ordered, time-boxed, rollback-aware. Complexity legend: **S** = ≤1 day · **M** = 2–4 days · **L** = 1–2 weeks · **XL** = 2–4 weeks.

## Phase 42B.1 -Peripheral repo bootstrap (S)

1. `mkdir ~/Work/btr/peripheral && cd ~ && forge init --no-commit peripheral` (M-style scaffold).
2. Add Solady submodule pinned to alm's current commit hash.
3. Emit `foundry.toml` (solc 0.8.29, via-ir, opt 200), `remappings.txt`, `.gitignore`.
4. Smoke `forge build` on empty src.

**Risk**: Solady commit drift between alm and pinned version. **Rollback**: rm dir; alm + dex untouched.

## Phase 42B.2 -Extract AS-IS modules (M)

5. Copy alm `AccessControl.sol` → `peripheral/src/access/AccessControl.sol`. Update import paths.
6. Copy alm `PriceProvider.sol` → `peripheral/src/oracle/PriceProvider.sol`. Add `IPriceProvider.sol` extracted iface.
7. Copy alm `PosMath.sol` → `peripheral/src/math/PosMath.sol`.
8. Copy dex `LibTimelock.sol`, `LibCast.sol`, `LibRescue.sol` → `peripheral/src/libs/`. Strip dex-specific imports (replace `IErrors` w/ peripheral `Err`).
9. Build `peripheral/src/errors/Errors.sol` = union of alm `Err` + dex `IErrors` selectors. Keep `Resource` enum from dex.
10. Port alm tests for AC + PriceProvider + PosMath; port dex tests for LibTimelock + LibRescue.
11. Goal: `forge test` 100% green; coverage ≥ source repos' coverage.

**Risk**: Errors merge introduces selector collisions (4-byte). **Mitigation**: pre-extraction selector audit; rename collisions in dex (less audit scope). **Rollback**: revert peripheral commits.

## Phase 42B.3 -Generalize libs + factory (M)

12. `LibTransientCache` -split oracle-specific fns into dex local extension; keep generic bitmask-slot derivation in peripheral.
13. `LibConstants` (peripheral) = `WAD/BPS/PBPS/HALF_PBPS/NATIVE` only. dex retains its `LibConstants` minus those (re-export).
14. `BaseFactory.sol` -abstract w/ `_cloneAndRegister(impl, ac)` helper. alm `Factory` will inherit; dex `PoolProxyFactoryV1` will inherit (CREATE3 path overrides clone).
15. `RoleBitmask.sol` -optional bitmask role extension consumed by dex modules needing pauser/oracle-pusher.

**Risk**: Generalization perturbs library inlining → bytecode growth in consumers. **Mitigation**: bench `forge build --sizes` pre/post; if any contract crosses 23kB threshold, defer that lib's generalization.

## Phase 42B.4 -alm consumes peripheral (M)

16. Add `peripheral/` as submodule to `alm/evm/lib/peripheral`.
17. Update alm `remappings.txt`: `@btr/peripheral=lib/peripheral/src/`.
18. Delete duplicates: alm `AccessControl.sol`, `PriceProvider.sol`, `Errors.sol`, `PosMath.sol`. Replace imports w/ `@btr/peripheral/...`.
19. Refactor alm `Factory.sol` to extend `BaseFactory`.
20. Refactor alm `Vault.salvage` to call `LibRescue` (drops local token-type branches, reduces `Vault` LOC).
21. Run full alm test suite (263 tests). **Acceptance**: zero regressions, all green.
22. Re-run Phase 16 audit-set static checks (Slither + manual checklist) -verify no new findings.

**Risk**: Solidity import-path edits introduce silent ABI drift. **Mitigation**: `forge inspect` ABI hash diff per contract; only AC/PriceProvider ABIs allowed to change (move). **Rollback**: git revert; submodule path unaffects on-chain deployments.

## Phase 42B.5 -dex consumes peripheral, swap AC (L)

23. Add submodule + remapping to `dex/contracts/`.
24. Replace `IErrors` inheritance across all dex contracts w/ `import {Err} from "@btr/peripheral/errors/Errors.sol"`. ABI surface drops `IErrors` selectors (now lib-level).
25. Migrate dex governance: `_s().owner` → `address ac` immutable on PoolProxyV1. Modifier reads `IAccessControl(ac).owner()` w/ `LibTransientCache` per-tx cache.
26. Replace dex inline-asm reentrancy w/ Solady `ReentrancyGuardTransient` (BaseV1 inherits).
27. Replace dex `LibTimelock` import path → peripheral.
28. Replace dex `LibRescue` import path → peripheral; drop dex-local copy.
29. dex factory (`PoolProxyFactoryV1`) extends `BaseFactory` for clone + register pattern; CREATE3 path retained.
30. Wire dex `OracleConfig.secondary` to optionally point at peripheral `PriceProviderOracleAdapter` -fallback when internal TWAP stale.
31. Run full dex test suite. Update tests for new owner-resolution path. **Acceptance**: zero behavioral regressions.

**Risk**: Per-pool storage migration for `_s().owner`. dex has live storage layout -changing PoolStorage = breaks deployed pools. **Mitigation**: keep `_s().owner` field in storage layout (deprecated, unused) to preserve slot offsets; new modifier reads `ac.owner()`. Document as Phase 42D cleanup post-migration.

**Risk**: Transient cache for `ac.owner()` -if AC.owner rotates mid-tx via reentrancy path, stale read. **Mitigation**: cache invalidates on tx boundary (transient); `ac.owner()` rotation requires owner signature so reentrancy-into-rotation is gov-only and safe.

**Rollback**: git revert dex commits; submodule still consumed by alm; on-chain deployments unaffected (refactor pre-deploy).

## Phase 42B.6 -Style parity refactor (L)

32. Apply Style Guide (doc 05) deltas to dex modules: custom-error doctrine, named selector constants, transient reentrancy via Solady, comment density, EIP-170 budget passes.
33. Per-module: bench `forge inspect <Contract> deployedBytecode | wc -c`. Modules >23kB queued for split (e.g. `AdminV1` → AdminConfig + AdminTimelock if needed).
34. Run dex test suite + benchmark gas table; require gas regression ≤ 3% per hot-path.

**Risk**: aggressive minimization breaks subtle invariants. **Mitigation**: each refactor is one-module-per-PR; mandatory diff review against pre-refactor `forge test --gas-report`.

## Phase 42B.7 -ABI regen + sdk + front sync (S)

35. Regenerate ABIs: peripheral → `~/Work/btr/sdk/src/abis/peripheral/`; alm → `sdk/src/abis/alm/`; dex → `sdk/src/abis/dex/`.
36. Bump `@btr-protocol/sdk` minor version.
37. front (`~/Work/btr/front`) updates `package.json` sdk version; smoke svelte-kit dev.
38. swap (`~/Work/btr/swap`) revisited only if uses AC ABI directly (likely no).

**Risk**: Front compile errors if AC ABI shape changed (selector parity preserved → low risk). **Rollback**: pin previous sdk version.

## Phase 42B.8 -Deploy + audit handoff (XL)

39. Testnet redeploy (Sepolia + chain-of-record) -fresh AC + PriceProvider + alm + dex.
40. Run integration suite (keepers + front e2e).
41. Audit handoff per doc 06.

**Risk**: Mainnet rotation requires gov 2-step + 7d/14d timelocks. Coordinate with active deployments.

## Total complexity

8 phases · 1 S · 4 M · 3 L · 1 XL ≈ **6–10 weeks** wall-clock w/ single Solidity engineer; 4–6 weeks w/ paired review.

## Hard rollback gates

- Any phase failing tests **MUST** revert before next phase.
- Phase 42B.5 (dex AC swap) is the irreversible bridge -past it, dex can't trivially re-vendor local AC. Sign-off required from gov-lead before merge.
- On-chain rollback: each phase pre-deploy. Post-deploy rollback = redeploy previous AC + factory queueAC migration (7d timelock).
