## Diff at a glance

| Aspect | alm | dex |
|---|---|---|
| Pragma | `^0.8.29` | `^0.8.33` |
| Access ctrl | external `AccessControl` SSoT | `_s().owner` per-pool storage, `Ownable.Unauthorized()` |
| Errors | shared `Err` lib | `IErrors` interface inherited |
| Reentrancy | Solady `ReentrancyGuardTransient` | inline assembly tload/tstore |
| Timelock | per-resource fields | unified `LibTimelock` (96-bit packed) |
| Pricing | external `PriceProvider` (Chainlink) | dual: `InternalOracleV1` (push TWAP) + `ExternalOracle` |
| Deployment | `LibClone` (1167) + `Factory` | CREATE3 + `PoolProxyFactoryV1` + diamond modules |
| Storage | direct slots in `Vault` | ERC-7201 namespaced (`CORE/STAKING/DISTRIBUTOR/ORACLE/RESCUE/FLOW_GUARD`) |
| Upgrade path | rotate via Factory | per-selector module timelock w/ trusted-codehash gate |

Style asymmetry = motivation for shared-peripheral extraction (doc 02) + dex parity refactor (doc 05).
